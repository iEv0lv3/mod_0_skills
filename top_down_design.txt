Become a Turing student

1 Choose Turing
  1.1 Define what is important to me in a software development program
  1.2 Research software development programs
    1.2.1 Google software development programs
    1.2.2 Open top results
    1.2.3 Read information about each program
      1.2.3.1 Select top choices for further investigation
    1.2.4 Research reviews of each top choice
      1.2.4.1 Google reviews for top choices
      1.2.4.2 Read reviews for each top choice
  1.3 Compare top choices of software development programs
    1.3.1 Define what program is most in line with goals
    1.3.2 Define what programs are economically viable
    1.3.3 Define what program is most in line with culture preferences
2 Apply to Turing
  2.1 Submit required materials
    2.1.1 Gather required materials
      2.1.1.1 Resume
      2.1.1.2 GitHub profile
      2.1.1.3 LinkedIn profile
    2.1.2 Write an introduction
      2.1.2.1 Define in own words why Turing
  2.2 Complete logic questions
    2.2.1 Gather materials to solve questions
      2.2.1.1 Notebook
      2.2.1.2 Pencil
      2.2.1.3 Eraser
    2.2.2 Solve each question
      2.2.2.1 Draw out each questions parameters
      2.2.2.2 Draw out each questions data
      2.2.2.3 Draw out conclusive data from answers
      2.2.2.4 Using algorithmic thinking define what I know
      2.2.2.5 Fill in the data as it is defined
      2.2.2.6 Submit answer
3 Turing Interview
  3.1 Schedule interview
    3.1.1 Use provided link to find available interview times
    3.1.2 Sign up for an interview date and time
    3.1.3 Define in-person interview or remote interview
  3.2 Prepare for interview
    3.2.1 Study
      3.2.1.1 Pick core concepts to study
        3.2.1.1.1 OOP
        3.2.1.1.2 Ruby
        3.2.1.1.3 Rails
      3.2.1.2 Deep dive study into Turing the business
      3.2.1.3 Deep dive study into Turing's culture
  3.3 Attend interview
    3.3.1 Travel to Turing campus
    3.3.2 Enter Turing and find interviewer
    3.3.3 Sit down for interview
      3.3.3.1 Solve logic problem
      3.3.3.2 Discuss the program and expectations
      3.3.3.3 Ask questions
  3.4 Complete interview
4 Accepted into Turing

# And that's 25 minutes :)
